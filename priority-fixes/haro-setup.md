# HARO & Journalist Platform Setup Guide

## 1. Help A Reporter Out (HARO / Connectively)
**URL**: `https://connectively.us` (Formerly HARO)
**Goal**: Access the 3x daily query feed from top-tier publications.

### Setup Instructions
1. Navigate to the registration page and sign up as a **Subject Matter Expert / Source**.
2. Complete your profile using Denisse Martinez's credentials (Principal Advisor, Nearshore Navigator).
3. **Keyword Alerts**: Set up daily digest routing for the following keywords:
   - `"nearshoring"`
   - `"manufacturing Mexico"`
   - `"supply chain"`
   - `"maquiladora"`
   - `"contract manufacturing"`
   - `"industrial real estate"`
   - `"reshoring nearshoring"`
   - `"Mexico manufacturing"`

## 2. Qwoted
**URL**: `https://www.qwoted.com`
**Goal**: A modern alternative to HARO with direct journalist messaging capabilities.

### Setup Instructions
1. Create a free **Expert** account.
2. Link the Nearshore Navigator LinkedIn company page and Denisse's personal LinkedIn profile.
3. Add "Supply Chain", "Logistics", and "International Trade" to your area of expertise tags.
4. Set email notifications to "Immediate" for matches to ensure response within the 60-minute golden window.

## 3. Featured.com (Formerly Terkel)
**URL**: `https://www.featured.com`
**Goal**: Target specific B2B queries where your answers are published directly as expert roundups.

### Setup Instructions
1. Create an expert profile.
2. Under industry focus, select "Manufacturing," "Logistics & Supply Chain," and "Real Estate."
3. Fill out the "Topics I Can Discuss" section with detailed long-tail capabilities (e.g., "USMCA compliance", "Shelter company setup", "Total landed cost analysis").
