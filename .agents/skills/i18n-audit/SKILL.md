# Skill: i18n Translation Audit

## Overview

This skill provides rules for auditing and refining business-critical translations across Spanish, Portuguese, Chinese, Korean, and English. It ensures region-specific register, terminology accuracy, and cross-language consistency.

## Language Rules

### Spanish (ES) - Mexico

- **Register**: Use "usted" imperative for CTAs (Agende, Reserve).
- **Terminology**: T-MEC, Empresa Shelter, Nave Industrial, Cruce Fronterizo.
- **Formatting**: Include opening ¿ and ¡. Avoid "vosotros" or Peninsular Spanish slang.

### Portuguese (PT) - Brazil

- **Register**: Warm and direct (Fale com a Denisse).
- **Terminology**: Nearshoring, Shelter, Cadeia de Suprimentos, Galpão Industrial.
- **Formatting**: Use Brazilian gerunds (fazendo) and pronouns. Convert metrics to m².

### Simplified Chinese (ZH) - Mainland China

- **Register**: Extremely formal "您" (nín). Use 贵公司 for "your company".
- **Terminology**: 中国加一战略 (China Plus One), 工业园区, 美墨加协定 (USMCA).
- **Formatting**: Verify Simplified characters only. Use 万/亿 for large numbers.

### Korean (KO) - South Korea

- **Register**: Formal 합쇼체 (-습니다/-ㅂ니다). Use 귀사 for "your company".
- **Terminology**: 니어쇼어링, 마킬라도라 (explain on first use), 인건비.
- **Formatting**: Korean date format (YYYY년 MM월 DD일).

### English (EN) - US

- **Register**: Confident boutique consultancy.
- **Terminology**: American spelling (labor, analyze).
- **Formatting**: Title Case for all headers. Spelled-out acronyms on first use (FAST, NNN).

## Cross-Language Checks

- Verify URL prefixes (`/[lang]/`).
- Ensure addresses and phone numbers are consistent.
- Sync tool link logic (Calendly, Calculator).
- **WhatsApp**: Ensure WhatsApp CTA is present and correctly translated in all versions (no WeChat/KakaoTalk).
