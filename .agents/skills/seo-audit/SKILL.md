---
name: seo-audit
description: Audits a URL against SERP competitors for ranking potential. Use when evaluating SEO gaps.
---

## Steps

1. Search the given keyword in Chrome via Google.
2. Identify top 3 ranking URLs.
3. Compare: H1-H3 structure, content depth, schema markup, author signals, page speed, backlink signals visible on-page.
4. Output a side-by-side gap analysis in markdown.
